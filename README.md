# FlipChange

Premium demo crypto exchange interface with live crypto charts, demo trading, balance, portfolio, and trade history.

## What is inside

- BTC, ETH, TON, SOL, BNB, XRP, DOGE, ADA markets.
- Live prices through Binance WebSocket with a demo fallback stream.
- Canvas charts, order book, trade tape, demo buy/sell ticket.
- Demo balance, portfolio allocation, and trade history.
- Static HTML/CSS/JS app, no build step required.

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.
